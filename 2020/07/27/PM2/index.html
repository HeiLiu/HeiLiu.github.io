pm2 是 node 进程管理工具，可以利用它来简化很多 node 应用管理的繁琐任务，如性能监控、自动重启、负载均衡等。

是的，就这么简单，一条命令就能启动一个 HTTP 服务器。由于这个 HTTP 服务器使用的是 NodeJS 实现，因此性能同样非常优异，在大部分情况下足够使用。如果是负载非常大的业务，一般也不会考虑使用 PM2，而会使用更具扩展性的Kubernetes。
