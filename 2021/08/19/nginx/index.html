Nginx

2021年08月19日 92 字 大概 1 分钟
  • 高性能的 web 服务器，开源免费
  • 一般用于静态服务、负载均衡
  • 反向代理
  • 访问日志 记录用户访问的每一条信息

测试配置文件格式是否正确

Mac 配置文件 /usr/local/etc/nginx/nginx/conf
Windows 配置文件 C:/nginx/conf/nginx.conf

启动 nginx

nginx

重启

nginx -s reload

停止

nginx -s stop$$
