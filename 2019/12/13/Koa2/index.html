Koa2 原生支持 async/await
// async/await 要点
// 1. await 后面可以追加 promise 对象，获取 resolve 的值
// 2. await 必须包裹在 async 里面，二者配套使用
// 3. async 函数执行返回也是一个 promise 对象
// 4. 可以使用 try-catch 捕获 promise 中 reject 的错误
